## Descripción del proyecto

Adopta Mascotas es una aplicación móvil desarrollada en Android con Jetpack Compose. Su objetivo es simular una plataforma básica de adopción de mascotas, donde el usuario puede consultar mascotas disponibles, revisar su información detallada y enviar una solicitud de adopción.

La aplicación cuenta con un flujo de navegación completo entre pantallas: inicio, lista de mascotas, detalle de mascota, formulario de adopción y perfil/resumen. Además, aplica conceptos de Programación Orientada a Objetos mediante el uso de clases como `Mascota` y `SolicitudAdopcion`, las cuales permiten organizar la información principal del sistema.

## Funcionalidades principales

- Visualización de mascotas disponibles.
- Consulta de detalle de cada mascota.
- Formulario para enviar solicitudes de adopción.
- Resumen de solicitudes enviadas.
- Navegación funcional entre pantallas.
- Interfaz moderna utilizando Jetpack Compose y Material Design 3.

## Tecnologías utilizadas

- Kotlin
- Android Studio
- Jetpack Compose
- Material Design 3
- Navigation Compose
- Git y GitHub

## Flujo de la aplicación

El usuario inicia en una pantalla principal donde se presenta la app. Desde ahí puede acceder a la lista de mascotas disponibles. Al seleccionar una mascota, se muestra una pantalla de detalle con su información y disponibilidad. Si la mascota está disponible, el usuario puede llenar un formulario de adopción. Finalmente, las solicitudes enviadas se muestran en la pantalla de perfil o resumen.
